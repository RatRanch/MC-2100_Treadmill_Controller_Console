# MC-2100 Treadmill Controller Console
Arduino console controller for the popular MC-2100 fitness motor controllor board used in Icon, NordicTrack and Proform exercise equipment.

This program runs on an Arduino Uno Rev3 or compatible board supporting 5V TTL levels.  It provides a basic set of controls to adjust the belt speed and incline angle.  A tachometer input is used to capture actual belt speed.

Please see the wiring diagram and build details at https://jimandnoreen.com/?p=484
